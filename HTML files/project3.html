Hostel Management System Software (HMSS)

This project is aimed to developing an online Hostel Management System Software (HMSS).

The Hostel Management Software which is created for managing different activities in the hostel. This project is expected to limit human works and make hostel allocation much easier for student and hostel administrators with the help of the web application to hostel, naturally select the student from the waiting list and mess billing, out-pass generation, complaint registration, and so forth

Hostel Managements System is intended for Hostel (like schools, colleges)

There will be pre-characterized criteria for the Reserve to the hostels.

If the students are qualified then they are designated to the hostel Room.
